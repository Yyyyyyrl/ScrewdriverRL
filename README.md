# ScrewdriverRL

Continuous screwdriver turning with dexterous hands in Isaac Lab using teacher-student reinforcement learning.

## Layout

- `screwdriver_rl/`: package source
- `scripts/`: training, evaluation, play, and smoke-test entry points
- `tests/`: unit tests

## Development

Install the project in editable mode from this directory:

```bash
pip install -e .
```

Run the test suite:

```bash
pytest
```
